catchment_data_api.cohorts

This module contains the Cohorts class, which is used to manage and calculate livestock cohort populations within a specified catchment area.

Computes the population of each cohort within the specified catchment area. This method calculates the population for both cattle and sheep cohorts based on the catchment’s raw herd data and the computed cohort coefficients.
