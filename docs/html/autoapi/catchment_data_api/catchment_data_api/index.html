catchment_data_api.catchment_data_api

A Python package for interfacing with various catchment-related data. This includes accessing, formatting, and aggregating data related to livestock, forests, cultivated areas, peat, and LUCAS survey data within different catchment areas.

This module provides a class, CatchmentDataAPI, that acts as an interface to the catchment data. It allows users to retrieve data for livestock, forests, cultivated areas, peat, and LUCAS survey data for all catchments or for a specific catchment.

The class also provides methods to calculate the total population of livestock by type for a specified catchment and to format catchment names for consistency.
